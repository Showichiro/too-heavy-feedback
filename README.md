# too-heavy / Feedback

[**too-heavy**](https://github.com/Showichiro/too-heavy) は、文明崩壊後の世界を歩き続ける一人用ウェブゲームです。プレイヤーは「進む」操作を持たず、限られた資源の中で **何を捨てるか** だけを判断し、どこまで遠くへ到達できるかを競います。

このリポジトリは、too-heavy 本体への **バグ報告・機能要望・質問・雑談** を受け付けるためのフィードバック専用リポジトリです。本体のコードは別リポジトリ (private) にあるため、ここにはコードは含まれません。

## ゲームを遊ぶ

- 本体: <https://github.com/Showichiro/too-heavy>
- プレイ: 公開後にリンクを追記予定

## フィードバックの送り方

| 内容 | 場所 |
| --- | --- |
| バグ報告 | [Issues](https://github.com/Showichiro/too-heavy-feedback/issues/new?template=bug_report.yml) |
| 機能要望・改善提案 | [Issues](https://github.com/Showichiro/too-heavy-feedback/issues/new?template=feature_request.yml) |
| 質問・遊び方の相談 | [Discussions / Q&A](https://github.com/Showichiro/too-heavy-feedback/discussions/categories/q-a) |
| 雑談・感想・スコア共有 | [Discussions / General](https://github.com/Showichiro/too-heavy-feedback/discussions) |
| バランス・難易度の意見 | [Discussions / Ideas](https://github.com/Showichiro/too-heavy-feedback/discussions/categories/ideas) |

迷ったら Discussions に投げてもらって構いません。こちらで適切な場所に振り分けます。

## 報告時のお願い

- **再現手順** をできるだけ具体的に書いてください (どの版・どの端末・何をしたか)
- バランス・難易度に関する意見は、到達距離 / プレイ時間 / 使ったアイテムなどの **状況** を添えてもらえると助かります
- 同じ内容の Issue / Discussion がないか軽く検索してから投稿してください

## 行動規範

建設的なやり取りをお願いします。他者への攻撃的な発言・差別的な表現・スパムは削除し、悪質な場合はブロックします。
